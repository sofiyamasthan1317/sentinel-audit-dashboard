# Audit Log Dashboard — Frontend Application

React + TypeScript + Vite frontend for the Gidy Audit Log Technical Assessment.

## Quick Start

```bash
# Install dependencies
npm install

# Start Vite dev server
npm run dev

# Production Build
npm run build
```

## Environment Configuration
Set in `.env`:
```env
VITE_API_BASE_URL=http://localhost:5000/api
```
